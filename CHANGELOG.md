# История изменений (Changelog)

Все заметные изменения в проекте "Progress Atlas" будут документироваться в этом файле.

## [1.1.0] - Firebase & RBAC Update
### Добавлено
- **Облачная база данных (Firestore):** Прогресс (выполненные задачи, оценки понимания, заметки) теперь сохраняется в облаке Firebase, обеспечивая синхронизацию между устройствами.
- **Авторизация (Firebase Auth):** Внедрен вход через Google-аккаунт.
- **Ролевая модель (RBAC):** Добавлен режим администратора (Godmode) с привязкой к конкретному email (`genialnee@gmail.com`). 
- **Режим "Только чтение":** Для всех неавторизованных пользователей интерфейс переходит в режим просмотра (read-only) с визуальной индикацией (неактивные поля, изменение курсора).

### Исправлено
- **Правила безопасности Firestore:** Настроены строгие правила `firestore.rules`, разрешающие запись в базу только администратору и защищающие данные от несанкционированного изменения.
- Устранен баг с кэшированием окна авторизации Google (добавлен параметр `prompt: 'select_account'`).
- Удалена устаревшая логика сохранения данных в `localStorage`.

## [1.0.0] - Initial Release
### Добавлено
- Базовый интерфейс приложения (Progress Atlas).
- Статический учебный план (Learning Plan).
- Консоль для ввода команд (Godmode).
- Поддержка двух языков (RU / EN).
